TRABAJO INVESTIGACIÓN WEB

Abstract

En esta asignatura de TIC II (Tecnologías de la información y la comunicación de segundo de bachiller) se ha propuesto hacer este trabajo para buscar información acerca de la WEB e internet. Este trabajo está planteado con la finalidad de aprender cómo funciona y para qué sirve.

En los primeros apartados se puede encontrar información sobre internet explicando así las principales diferencias de internet con la web, en estos apartados también se pueden observar cómo se profundiza en los aspectos característicos de la WEB tanto la WEB 1.0, 2.0 com la WEB 3.0.

Por último, se da un ejemplo de red social Facebook en la que se explica quien fue su creador dando también información sobre sus funcionalidades, aparte del impacto tan grande que supuso para la WEB 2.0 y el mundo online.

Tras la búsqueda de toda esta información sobre la WEB e internet se puede llegar a la conclusión de que la WEB va evolucionando a lo largo de los años y ya se pueden encontrar las principales diferencias entre la WEB e internet.
